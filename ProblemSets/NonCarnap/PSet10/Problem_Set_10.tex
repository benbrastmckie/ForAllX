\textbf{Argument:} Consider the following line of reasoning.

  \begin{itemize}
    \item[-] Hesperus is rising.
    \item[-] Hesperus is Phosphorus.
    \item[$\therefore$] Phosphorus is rising.
  \end{itemize}




\textbf{Questions:} (1 point each)

\begin{enumerate}
  \item Regiment the argument above in QL.
    \begin{enumerate}
      \item If the argument is valid, provide a semantic proof of its validity.
      \item If the argument is invalid, provide a countermodel.
    \end{enumerate}  
  \item Regiment the argument above in QL$^=$.
    \begin{enumerate}
      \item If the argument is valid, provide a semantic proof of its validity.
      \item If the argument is invalid, provide a countermodel.
    \end{enumerate}  
  \item Which language provides a better regimentation and why?
\end{enumerate}

\bigskip




\textbf{Hints:} Remember...

\begin{enumerate}
  \item Officially the extension of $1$-place predicates is a set of $1$-tuples.
  \item Do not skip steps in your semantic proofs.
\end{enumerate}
